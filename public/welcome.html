Admin Interface

Communicator provides an Admin interface. When you set up the environment you gave the 'admin' account a password. Use the Admin account to add and remove user accounts, and to set the 'profile' of each account. A profile determines the capabilities granted to each user.

/admin

The defaults are:

  • username: admin
  • password: whatUsee

User Interface

Users log into the Front-Page of Communicator at the root url. Each user should log in with their Name and Password.
